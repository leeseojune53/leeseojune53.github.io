### 🔓 버킷 정책 변경

버킷 생성 후 버킷에서 권한탭을 들어가서 정책에서 편집을 누른 후

```
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::music-ward/*"
        }
    ]
}
```

를 기입해주고 변경 사항 저장을 눌러준다.
